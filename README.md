# Site vitrine — Bistro chic (démo)

Site statique HTML/CSS/JS (GitHub Pages compatible).
Aucun PDF / aucune image dans le repo : les photos sont chargées via URLs (ex. Unsplash).

## Structure
- `index.html` : accueil
- `menu.html` : menu dynamique (JS, données intégrées)
- `contact.html` : contact + réservation + carte
- `css/main.css` : design system “bistro chic”
- `js/main.js` : nav mobile + ouvert/fermé + génération du menu
- `legal/` : mentions légales + confidentialité

## Personnalisation client
Tout est dans `js/main.js` (objet `SITE`) :
- nom, adresse, téléphone, email
- horaires
- menu
